# C# SignalR实时应用

## 描述
使用C#和SignalR技术创建一个实时Web应用，支持双向通信和实时数据更新。

## 要求
1. 使用C#、ASP.NET Core和SignalR
2. 实现以下功能：
   - 实时消息传递
   - 用户在线状态管理
   - 群组通信
   - 实时数据更新和同步
   - 客户端到服务器和服务器到客户端的事件
   - 连接管理和重连逻辑
   - 消息持久化
   - 用户认证和授权
3. 创建一个示例应用（如聊天应用、协作工具或实时仪表盘）
4. 实现适当的错误处理和日志记录
5. 添加单元测试
6. 优化性能和可扩展性
7. 提供客户端JavaScript/TypeScript代码

## 输入
无特定输入，需要从头创建ASP.NET Core SignalR项目。

## 期望输出
1. 完整的ASP.NET Core SignalR应用源代码
2. SignalR Hub实现
3. 客户端JavaScript/TypeScript代码
4. 用户界面实现
5. 消息持久化实现
6. 测试代码
7. 项目结构说明
8. 构建和运行指南

## 评估标准
1. SignalR技术的正确使用
2. 实时通信的可靠性
3. 连接管理和错误处理
4. 性能和可扩展性考虑
5. 安全性实现
6. 代码组织和结构
7. 测试覆盖率
8. 文档的清晰度和完整性
