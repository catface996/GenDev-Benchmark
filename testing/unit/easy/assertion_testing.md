# 断言测试

## 描述
学习如何使用各种断言方法编写有效的单元测试，验证代码行为和结果的正确性。

## 要求
1. 理解断言测试的概念和重要性
2. 实现以下断言测试场景：
   - 相等性断言测试
   - 布尔条件断言测试
   - 集合和数组断言测试
   - 异常断言测试
   - 对象属性断言测试
   - 自定义断言测试
3. 使用测试工具和框架：
   - 单元测试框架的断言库
   - 扩展断言库
   - 断言失败处理
   - 断言消息定制
4. 实现测试辅助功能：
   - 断言选择和组合
   - 断言消息设计
   - 断言结果验证
5. 编写清晰的测试文档和注释

## 输入
被测代码和预期行为规范。

## 期望输出
1. 断言测试代码
2. 断言选择和组合示例
3. 测试执行结果
4. 断言失败分析
5. 测试文档和注释

## 评估标准
1. 断言的正确选择和使用
2. 断言测试的完整性和有效性
3. 断言消息的清晰性和帮助性
4. 测试代码的可读性和可维护性
5. 测试文档的清晰度
